# Medicare Part D Prescribers - By Geography & Drug

## Dataset

The dataset used in this project exceeds GitHub’s file size limits.

This project uses the Medicare Part D Prescriber Geographic Data (2017–2023).

Due to file size restrictions, the dataset is not stored directly in this repository.

Original data source:
https://data.cms.gov/provider-summary-by-type-of-service/medicare-part-d-prescribers/medicare-part-d-prescribers-by-geography-and-drug

## Data Preparation

The working dataset was created by combining yearly CMS geographic files using R Studio.

See `RScript_Combining_geo_files.R` for full details.
